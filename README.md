# highlightjs-sap-abap

SAP ABAP highlighting for highlight.js

## Build distributable

- install this repo in folder extra of [highlightjs](https://github.com/highlightjs/highlight.js)
- run the build tool from the main highlightjs folder:

```sh
npm run build-cdn
```
